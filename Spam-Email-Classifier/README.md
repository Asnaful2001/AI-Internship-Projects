# AI Spam Email Classifier

This project is an AI-based Spam Email Classifier developed using Machine Learning and Natural Language Processing (NLP).

## Project Overview

The model analyzes text messages/emails and predicts whether they are Spam or Ham (Not Spam).

## Technologies Used

- Python
- Pandas
- Scikit-learn
- NLP (CountVectorizer)
- Naive Bayes Algorithm
- Matplotlib

## Features

- Data Preprocessing
- Text Vectorization
- Machine Learning Model Training
- Spam Detection Prediction
- Accuracy Evaluation
- Data Visualization

## Conclusion

This project helped in understanding Machine Learning workflows, NLP techniques, and AI-based text classification systems.
